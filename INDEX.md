## 🎨 Personnalisation

### Couleurs
Modifiez `styles.css` :
```css
:root {
    --primary-color: #0a0e27;
    --accent-color: #6366f1;
}
```

### Raccourcis
Modifiez `newtab.html` ou `config.json`

### Animations
Modifiez les valeurs `transition` et `animation` dans `styles.css`
